Setup

Next: Configuration »

Requirements

  1. Git
  2. Ruby 1.9.3 or greater using either RVM or rbenv
  3. ExecJS supported JavaScript runtime

To check your ruby version, run $ ruby --version.
If your version is at least Ruby 1.9.3, then you’re good to go. Otherwise, install or update your ruby version with RVM or rbenv.

Set Up Showcase

Download or clone from Github

1$ git clone https://github.com/femmestem/Showcase.git showcase
2$ cd showcase

Install dependencies
Note: You will need to do this every time you update your version of Ruby.

1$ gem install bundler
2$ rbenv rehash # rehash rbenv to run bundle command, skip if using RVM
3$ bundle install

Install the default Showcase theme

1$ rake install

To verify the install was successful and preview your new site

1$ rake preview
